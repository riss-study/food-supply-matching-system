### 2.2 게시글 유형 및 패턴 분석

#### 2.2.1 의뢰글 (식품 제조 공장 구해)
**일반적인 포함 정보:**
- 제품명/카테고리
- 생산량 (MOQ)
- 희망 단가
- 배송/납기 요구사항
- 원재료 제공 여부
- HACCP 인증 필요 여부
- 지역 선호도
- 연락처

**예시 패턴:**
```
[의뢰] 떡볶이 소스 제조 공장 구합니다
- 제품: 즉석 떡볶이 소스
- 월 생산량: 5,000kg
- 희망 단가: 3,000원/kg
- HACCP 필수
- 원재료: 업체 제공
- 지역: 경기도 선호
- 연락처: 010-XXXX-XXXX
```

#### 2.2.2 수주글 (제조사_수주요)
**일반적인 포함 정보:**
- 업체명
- 보유 설비/인증
- 전문 분야
- 수용 가능 생산량
- 위치
- 연락처
- 포트폴리오/사례

**예시 패턴:**
```
[수주] 충청북도 식품제조업체입니다
- 업체명: XXX식품
- 인증: HACCP, ISO22000
- 전문: 소스류, 장류
- 생산량: 월 10톤 가능
- 위치: 충북 청주
- 연락처: 043-XXX-XXXX
```

#### 2.2.3 정보공유글
- 제조 노하우 공유
- 규제/법률 정보
- 시장 동향
- 실패/성공 사례

**개요**
- 북미 식품 계약 제조 전문 AI 플랫폼

| 기능 | 설명 |
|-----|------|
| AI-RFQ Matching | AI 기반 견적의뢰서 매칭 시스템 |
| Confidential Connection | 비밀 유지 연결 |
| Thousands of Providers | 수천 개의 제조업체 DB |
| CPA 인증 | 미국 계약포장협회 공식 인정 |

**매칭 프로세스**
1. 브랜드/프라이빗 레이블이 RFQ 작성

5. 브랜드가 응답 선택

- 브랜드/Retailer: 무제 사용

#### Priority 1 (필수)
1. **스마트 매칭 엔진**: 요구사항 기반 제조사 추천
   - 입력: 제품명, 수량, 예산, 인증요구사항
   - 출력: 적합도 순위별 제조업체 목록
   - 기술: 머신러닝 기반 scoring 알고리즘

2. **게시글 자동 분류**: 의뢰/수주 자동 구분 및 태깅
   - 입력: 게시글 제목+본문
   - 출력: 카테고리, 키워드 태그, 긴급도
   - 기술: 텍스트 분류 (Text Classification)

3. **챗봇 어시스턴스**: 기본 문의 자동 응답
   - 24/7 FAQ 응답
   - 의뢰/수주 프로세스 안내
   - 기술: LLM (대형 언어 모델)

#### Priority 2 (권장)
1. **견적 AI 예측**: 시장 데이터 기반 예상 견적
   - 입력: 제품 정보, 수량, 원재료
   - 출력: 예상 단가 범위
   - 기술: 회귀 분석, 시장 데이터 학습

2. **문서 OCR**: HACCP 문서, 영업신고증 자동 확인
   - 입력: 업로드된 문서 이미지/PDF
   - 출력: 인증 정보 자동 추출 및 검증
   - 기술: Computer Vision + NLP

3. **이상 징후 탐지**: 사기성 게시글 자동 필터링
   - 입력: 게시글 내용, 작성자 이력
   - 출력: 위험도 점수
   - 기술: Anomaly Detection

#### Priority 3 (향후 고려)
1. **수요 예측 모델**: 시즌별/트렌드별 생산량 예측
2. **자동 번역**: 글로벌 매칭 지원
3. **음성 인식**: 전화 상담 자동 기록
4. **품질 예측**: 제조업체 품질 예측 모델
